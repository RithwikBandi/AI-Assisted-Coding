## Folder Description

**Docx/**  
Contains editable and working documents used while preparing the assignments. These files may include draft versions, working copies, and intermediate formats such as `.docx`, `.pages`, and exported PDFs.

**PDF/**  
Contains the final, submission-ready PDF versions of the assignments. These files represent the finalized outputs intended for evaluation and record-keeping.

**Question/**  
Contains the original lab assignment question documents provided as part of the coursework. These files serve as references for understanding the problem statements and requirements.

---

## Purpose of This Repository

- Maintain a clean and organized record of lab assignments
- Separate original questions from working files and final submissions
- Track progress and submissions using GitHub version control
- Demonstrate the use of AI-assisted tools in coding and documentation workflows

---

## Usage

This repository is intended for academic lab assignment management, version control for coursework documents, easy access to questions, drafts, and final submissions, and backup/reference for completed lab work.

---

## Notes

- File names follow assignment numbering for easy identification
- PDFs in the `PDF` folder should be treated as final versions
- Working documents in the `Docx` folder may change over time
- Question files in the `Question` folder should remain unchanged for reference
